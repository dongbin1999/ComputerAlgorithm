## 측정방법

사각형의 크기는 400*400으로 (반지름의 길이 200)으로 근사값을 측정하였고,

 자료형은 double을 사용하였습니다.



사이즈가 작은 것 같아서 다음과 같이 세 경우로 나눠 각각 1000만번 씩 점을 찍었습니다.

1. 원의 경계점까지 포함해서 카운트한 경우
2. 원의 경계점에 점이 찍힐 때, 50%의 확률로 카운트 한 경우
3. 원의 경계선은 카운트에 포함하지 않은 경우



## 결과
